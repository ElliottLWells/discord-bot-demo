# Discord Bot Demo

A beginner-friendly Discord bot built with Python and `discord.py`. This project demonstrates basic bot setup, slash commands, embeds, buttons, dropdown menus, and reaction roles.

## Features

- Secure token loading with `.env`
- Slash command syncing for a test server
- Basic slash commands
- Embed message demo
- Button interaction demo
- Dropdown/select menu demo
- Reaction role demo

## Technologies Used

- Python
- discord.py
- python-dotenv
- Git/GitHub
